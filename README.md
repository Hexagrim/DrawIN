# DrawIN

This is a 2d game about putting a ball in a jar by drawing physics shapes to guide and ball or the environment.

## Controls

Use right mouse button to drag and draw, when overlapping with ground or the nondraw area it stops drawing.

## Levels

The game on its own has about 14 levels for showcase and has a special feature of loading, creating and exporting levels by yourself as a json file which can be edited visually in game.

## Thats it. Have fun, make levels and maybe share it with your friends!
